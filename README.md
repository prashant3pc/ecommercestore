# Edgeoft Product Listing

This is a **Next.js** e-commerce product listing project, built with **React**, **Tailwind CSS**, and **TypeScript**. It demonstrates a product listing page that fetches and displays products from an API, with a responsive and modern UI.

## Live Demo

Check out the live deployment on Vercel: [Ecommercestore Live](https://ecommercestore-8jb6.vercel.app/)

## Features

- Fetches product data from an API
- Displays products in a grid layout
- Responsive design with Tailwind CSS
- Built with **Next.js**, **React**, and **TypeScript**
- Easy to extend with filters, sorting, and search functionality

## Technologies Used

- [Next.js](https://nextjs.org/) – React framework for server-side rendering and static site generation
- [React](https://reactjs.org/) – Frontend library
- [TypeScript](https://www.typescriptlang.org/) – Static typing for JavaScript
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework
